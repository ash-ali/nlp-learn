## 斐波那契数列的几种优化实践

1 1 2 3 5 8 13 21....

1、递归
```python
def fib(n):
    if n<3:
        return 1
    return fib(n-2)+fib(n-1)
```


2、时间复杂度：n
```python
import numpy as np
import math
def fib(n):
    temp = np.zeros(n)
    temp[0] = 1
    temp[1] = 1
    for i in range(2,n):
        temp[i] = temp[i-2]+temp[i-1]
    return temp[n-1]
```
3、空间复杂度：1
```python
import math
def fib(n):
    a = math.sqrt(5.0)
    b = (1+a)/2                            
    c = (1-a)/2
    return (math.pow(b, n) - math.pow(c, n))/a + 0.5;
```
4、时间复杂度：1

公式法：通项公式Fib(n) = [(1＋√5)/2]^n /√5 － [(1－√5)/2]^n /√5
```python
import math
def fib(n):
    a = math.sqrt(5.0)
    b = (1+a)/2
    c = (1-a)/2
    return (math.pow(b, n) - math.pow(c, n))/a + 0.5;
```
